# Terminal Commands Reference
*GodTier Sprint — Week 1*

## Core Navigation Commands

| Command | What it means | Why you use it |
|--------|--------------|----------------|
| `cd foldername` | Go into a folder | Move forward, deeper into a folder |
| `cd ..` | Go up one level | Move backward, out of a folder |
| `cd "folder name"` | Go into a folder with spaces in its name | Same as above but quotes handle the spaces |
| `cd C:\full\path\here` | Jump directly to any location | When you know the exact address |
| `ls` | List everything in current folder | See what's inside where you are |
| `pwd` | Print Working Directory — shows your current location | Confirm exactly where you are |
| `mkdir foldername` | Make Directory — create a new folder | Build new folders |
| `clear` | Clear the terminal screen | Clean up when it gets cluttered |

## Key Rules
- Folder names with spaces always need quotes: `cd "Python projects 3"`
- `cd ..` goes up ONE level at a time — use it twice to go up two levels
- Always run `ls` when confused about where you are — see what's around you
- Always run `pwd` when lost — it shows your exact location
